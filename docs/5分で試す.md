# 5 分で試す（チュートリアル）

> **使い方のコツ**: Enter を押すと Claude にその時点の文章が **送信** されます。文章は 1 行で書ききってから Enter を押してください（途中で押してしまっても続きを書けば Claude が察してくれます）。

セットアップが終わったら、まずは **架空案件で 1 件**試して、出力イメージをつかむのが一番早いです。

## 0. 始める前のチェック

- [ ] [docs/セキュリティ.md](セキュリティ.md) を読みましたか？製造業の機密情報を扱うため、必読です
- [ ] [SUPPORT.md](../SUPPORT.md) を読みましたか？個別サポートはありません、自己解決ベースです
- [ ] Claude Pro 以上のプランで Claude Code にサインイン済みですか？

## 1. ENAA 資料を取得・配置

[reference/ENAA/README.md](../reference/ENAA/README.md) の手順で **5 ファイル** をダウンロードして配置してください。

理想形：

```
reference/ENAA/
├── README.md
├── mes_g25-029-1.pdf      ← 解説資料
├── mes_g25-029-2.xlsx     ← 標準業務一覧（コア）
├── mes_g25-029-3.pdf      ← クイックスタート
├── mes_g25-029-4.pdf      ← これまでの経緯
└── mes_g25-029-5.pdf      ← How to Use
```

PowerShell で配置確認：

```powershell
Get-ChildItem reference\ENAA\ | Select-Object Name, Length
```

5 ファイルが表示されれば OK。

## 2. Claude を起動して挨拶

```powershell
claude
```

起動しただけでは Claude は何も話しません。**こちらから先に話しかけます。**

```
こんにちは。
```

と打って Enter（最後の句点まで打つのがコツ）。

Claude が使い方を簡潔に説明して、メニューを出してくれます：

```
何をしますか？番号で答えてください。

  1) MES 仕様書を作る（ENAA 標準業務一覧をベースに 4 段階で対話）
       Stage 1: 調査星取表  →  Stage 2: 要求機能整理
       Stage 3: 期待効果・ROI  →  Stage 4: ロードマップ・RFP 一次稿生成
  2) ENAA 資料の確認
  9) 使い方を見る
  0) 終了
```

## 3. メニューから「1」を選ぶ

`1` を入力して Enter。

Claude が Stage 0（前提確認・案件メタ聴取）を始めます：

- Q: 案件名は？
  A: 例「テスト案件」
- Q: 対象工場のタイプは？（discrete = 組立加工系を想定）
  A: 例「discrete、自動車部品想定」
- Q: 期間は？
  A: 例「2027 年度から構築」
- Q: 予算感は？
  A: 例「1 億円程度」

最初は **架空の答え** で OK です。雛形がどう動くかを見るだけ。

## 4. Stage 1〜4 を進める（または途中で休む）

各 Stage の境目で Claude が「**ここで一旦休みますか？続けますか？**」と聞いてくれます。

途中で休む場合：
- ファイルは既に保存されているので、画面を閉じても続きから再開できます

続ける場合：
- 各 Stage で 1 問 1 答で答えていけば、最後に YAML と Markdown が生成されます

## 5. 出力ファイルを確認

`outputs/MES仕様書/<日付>_<案件名>.{yaml,md}` が生成されています。

```powershell
Get-ChildItem outputs\MES仕様書\
```

YAML は構造化データ、Markdown は人間可読の RFP 一次稿です。

**末尾に ENAA 著作権表示が自動で付記**されているはずです。これは ENAA 利用条件で必須なので、削除しないでください。

## 6. サンプルを参考にする

[outputs/MES仕様書/20260502_マルヨシ精工_MES導入仕様書.yaml](../outputs/MES仕様書/20260502_マルヨシ精工_MES導入仕様書.yaml) と [.md](../outputs/MES仕様書/20260502_マルヨシ精工_MES導入仕様書.md) は、架空企業（自動車向け精密切削部品メーカー）の例です。実案件の参考にどうぞ。
