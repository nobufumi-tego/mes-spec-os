# インストール手順（詳しい版）

## 必要なもの

- Windows 11（macOS / Linux でも動作可能だが本書は Windows 前提）
- インターネット接続
- 管理者権限（PowerShell を管理者として実行できること）
- Claude Pro 以上のプラン、または Anthropic API キー（**Claude Free での業務利用は非推奨**、[docs/セキュリティ.md](セキュリティ.md) 参照）

## 手順

### 1. PowerShell を管理者として実行で開く

スタートメニューから「PowerShell」を探し、右クリック →「管理者として実行」。

### 2. 実行ポリシー変更（初回だけ）

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

「Y」を押して確定。

### 3. 必要ツールをインストール

```powershell
winget install OpenJS.NodeJS.LTS Git.Git astral-sh.uv
```

3 つすべて入るまで待ってください。

- **Node.js LTS**: Claude Code の動作に必要
- **Git**: リポジトリ取得に必要
- **uv**: Python 環境管理（cache.py / extraction-recipes 用）

### 4. Claude Code をインストール

```powershell
npm install -g @anthropic-ai/claude-code
```

### 5. mes-spec-os を取得

```powershell
cd $HOME\Documents
git clone https://github.com/nobufumi-tego/mes-spec-os.git
cd mes-spec-os
```

### 6. ENAA 5 資料を取得・配置

[reference/ENAA/README.md](../reference/ENAA/README.md) の手順に従って、ENAA 公式サイトから 5 ファイルをダウンロードして `reference/ENAA/` 直下に配置してください。

> **重要**：ENAA 資料は本リポに含まれていません。利用者が ENAA 公式から個別取得する設計です（再配布回避＋ENAA 認知拡大）。

### 7. Claude を起動

```powershell
claude
```

初回はブラウザが開いて Claude のサインインを求められます。
Claude Pro 以上のアカウントでサインインしてください。

社内 Wi-Fi で繋がらない場合はスマホテザリングを試してください。
社内ネットワークのプロキシ設定が必要な場合は情シス担当者にご相談ください。

### 8. 「こんにちは。」と話しかける

サインインが完了して、文字を打ち込める入力欄が出たら成功です。
最初に **「こんにちは。」** と最後の句点まで打って Enter を押してみてください。Claude が使い方とメニューを案内します（何を打っても構いません。Enter を押した瞬間に送信されるので、**文として書ききってから** Enter）。

メニューから `1) MES 仕様書を作る` を選ぶと、ENAA 4 段階フローの対話が始まります。

## セキュリティの確認（**必読**）

業務利用前に必ず [docs/セキュリティ.md](セキュリティ.md) を確認してください。
特に以下：

- 利用するプラン（Claude Team / Enterprise が業務利用には推奨）
- 社内ルール・取引先要件（IATF 16949 等）に抵触しないか
- 機密データの伏せ字化指針
- 案件開始時 / 終了時のチェックリスト
