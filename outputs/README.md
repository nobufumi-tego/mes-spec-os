# outputs/

`/MES仕様書` コマンドが生成した仕様書 / RFP 一次稿が保存される場所です。

## ファイル構成

```
outputs/
└── MES仕様書/
    ├── <YYYYMMDD>_<案件名>.yaml   ← 構造化データ
    └── <YYYYMMDD>_<案件名>.md    ← RFP 一次稿（人間可読）
```

## ENAA 出典記載

すべての出力ファイルの末尾には ENAA 出典が**自動で付記**されます。
これは ENAA「MES/MOM 導入のための標準業務一覧」の利用条件（著作権表示必須）に基づく仕様で、削除しないでください。

## Git の取り扱い

デフォルトでは追跡対象です。ただし以下のような場合は個別判断で除外してください：

- 案件名や顧客名が機密の場合 → `.gitignore` に `outputs/<機密案件名>/` を追加
- ライン番号・製品コードなど現場固有情報が含まれる場合 → 同上
